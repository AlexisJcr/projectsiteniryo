Projet Gwendal Troadec


Gwendal a eu comme projet l'installation de ROS (Robot Operating System).
ROS permet de simuler le robot Ned².
La simulation permet aussi de gérer les conditions environnementales autour du robot.
Il permet également de faire des tests de simulation pour éviter tout dommage matériel sur le robot.

Sur la page ci-dessous, Gwendal a rédigé les instructions d'installation de ROS.
